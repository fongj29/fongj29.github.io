“Alt text is a written description of an image posted online. Alt text allows visual content to be accessible to people who are blind, low vision, or have certain cognitive disabilities.” Resource: Alt Text as Poetry  In case, you’ve never encountered alt text, here are the code snippets of how to add a photo and include alt text. Effective alt text is able to describe what is on the page and will be read aloud with a voiceover. In my case, I chose to embed my own audio clips as a voiceover to avoid the robotic monotone voice that would otherwise be used.
